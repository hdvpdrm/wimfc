# wimfc
This CLI tool follows only one goal - get the most frequently used command in ```fish``` session.

# how to start?
```bash
git clone https://github.com/hdvpdrm/wimfc.git
cd wimfc
g++ -o wimfc -std=c++14 main.cpp
chmod +x wimfc
./wimfc
```
